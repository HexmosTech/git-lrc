Why git-lrc

Reviews that catch what you miss

AI agents write code fast — and silently remove logic, change behavior and introduce bugs. git-lrc reviews every diff before it lands.

Runs on commit

Hooks into git commit and reviews the staged diff automatically — no extra step, no context switch.

Completely free

Unlimited AI reviews at no cost. Set it up once and review every commit, forever.

Runs locally

git-lrc runs as a local CLI by default — your code stays on your machine unless you submit a review.

Any AI provider

Bring your own connector — Gemini, Claude, OpenAI, DeepSeek and more, prioritized exactly how you want.

Severity-ranked findings

Issues are graded Critical, Error, Warning and Info, so you fix what matters before you ship.

Beautiful review UI

A clean local viewer with inline diffs, comments and a one-click hand-off to your coding agent.
